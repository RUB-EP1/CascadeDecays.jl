# Context Notes

This folder captures the current design context for `CascadeDecays.jl` based on:

- the requested study of `ThreeBodyDecays.jl`
- the agreed flat graph representation for cascade topologies
- the prototype sketch provided in the thread
- the unresolved question about masses on intermediate lines

Files:

- `01-threebodydecays-basis.md`: the concrete objects and patterns extracted from `ThreeBodyDecays.jl`
- `02-flat-topology-container.md`: the proposed flat topology container for the new package
- `03-relation-map-guidelines.md`: detailed rules for the line-vertex incidence map
- `04-prototype-notes.md`: mapping from the prototype code to the future package
- `05-open-design-questions.md`: unresolved issues and recommended decisions

These notes are meant to be the local project memory for the first implementation phase.
